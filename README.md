# Loketasisten

Loketasisten is an MVP web app that helps architects and bureaus draft answers for loket and permit dossiers faster using AI.

## Tech stack

- **Frontend**: Next.js 16 (App Router) + React + Tailwind CSS
- **Backend**: FastAPI (Python)
- **Auth**: Email/password with dummy token, ready for JWT later
- **Data/LLM**: In-memory “dummy DB” and dummy LLM responses (easy to swap for real systems later)
